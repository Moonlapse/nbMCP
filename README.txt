=== MCP 8.11 with Netbeans Project Files ===

=== Requirements ===

- Netbeans 8.0.1 with Java SE or EE
https://netbeans.org/downloads/

=== Install ===

- Download and install Netbeans

- Download and extract nbMCP repository to C:/nbMCP

- Navigate to C:/nbMCP folder

- Run decompile.bat (MAC use decompile sh)
---> This will take some time <---

- Open Netbeans

- Open Project -> C:/nbMCP/eclipse

- Open Client Project File

- MPC is now ready to edit with Netbeans
